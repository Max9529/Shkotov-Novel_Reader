# Novel Reader

🔗 **[Читати новели на GitHub Pages](https://Max9529.github.io/Shkotov-Novel_Reader/)**

Вебзастосунок-читалка для зручного читання перекладених новел у форматі Markdown. Створений як повністю статичний вебсайт (SPA), оптимізований для мобільних пристроїв та десктопів, що дозволяє комфортно читати улюблені твори без зайвих відволікань. Нові новели та розділи додаються до бібліотеки вручну автором проєкту.

## Як це працює та основні можливості

* **Імерсивний режим читання:** Панелі керування автоматично приховуються під час скролу вниз і з'являються при скролі вгору або за допомогою швидкого тапу по екрану.
* **Персоналізація тексту:** Зручна кастомна панель налаштувань для зміни шрифтів (зокрема Literata, Noto Serif, Merriweather), розміру шрифту, міжрядкового інтервалу та ширини текстового блоку.
* **Збереження прогресу:** Поточний прогрес читання, налаштування та закладки автоматично зберігаються локально у вашому браузері (`localStorage`).
